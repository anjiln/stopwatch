# Stopwatch

A feature-rich stopwatch application with lap recording and statistics tracking capabilities.

## Features

- **Time Tracking**: Accurate time measurement with hours, minutes, seconds, and centiseconds
- **Start/Stop**: Toggle between running and paused states
- **Lap Timing**: Record individual lap times
- **Statistics**: Automatically tracks and displays:
  - Fastest lap time
  - Slowest lap time
  - All recorded laps with their durations
- **Reset Function**: Clear all data and start fresh
- **Responsive Design**: Clean, card-based UI

## How to Use

1. Open `index.html` in a web browser
2. Click **Start** to begin timing
3. Click **Lap** to record a lap time (while running)
4. Click **Stop** to pause timing
5. Click **Reset** to clear all data and statistics
6. View your fastest and slowest laps in the stats section

## Files

- `index.html` - HTML structure with stopwatch layout
- `script.js` - JavaScript logic for timing, lap recording, and statistics
- `style.css` - Styling for the stopwatch interface
